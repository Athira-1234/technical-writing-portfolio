# Google Chrome Installation Guide

## Overview

This guide explains how to install Google Chrome on a Windows 11 computer.

## System Requirements

- Windows 11
- Internet connection
- Administrator access

## Installation Steps

1. Open Microsoft Edge.
2. Visit https://www.google.com/chrome
3. Click **Download Chrome**.
4. Open the downloaded installer.
5. Follow the installation instructions.
6. Click **Finish**.

## Troubleshooting

### Chrome won't download

- Check your internet connection.
- Restart your browser.
- Try downloading again.

### Installation failed

- Restart your computer.
- Run the installer as Administrator.

## Result

Google Chrome has been installed successfully.
